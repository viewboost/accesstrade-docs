### FR-T2-002: OAuth SSO Scalef

**Source:** Account Linking PRD — FR-002
**Priority:** Must Have

OAuth2 Authorization Code Flow theo Scalef SSO spec.

**Key Acceptance Criteria:**
- Generate `state` random, lưu Redis TTL 15 phút
- Redirect Scalef: `/oauth/authorize?provider=&client_id=...&scope=user_info&response_type=code&redirect_uri=...&state=...`
- Callback verify state (1 lần dùng, mismatch → 403)
- Token exchange backend-to-backend (`POST /oauth/token` với Basic auth `client_id:client_secret`)
- Parse response: `access_token`, `refresh_token`, `user_id`, `user_name`, `phone`, `email`
- Token store Redis với TTL theo `expires_in`
- **V1 simplification:** KHÔNG fetch CCCD, KHÔNG so khớp, KHÔNG check 1-1

**Full spec:** [Account Linking FR-002](prd-account-linking-2026-05-04.md#fr-002-đăng-nhập-scalef-oauth-sso)

### FR-T2-011: Liên kết thành công (V1 simplified)

**Source:** Account Linking PRD — FR-011 (modified)
**Priority:** Must Have

**V1 simplification:** Sau OAuth callback → lưu `scalef_user_id` thẳng vào users, không update profile.

**Key Acceptance Criteria:**
- Sau callback OAuth thành công + verify state:
  - Update `users` collection: `scalef_user_id`, `scalef_linked_at`, `scalef_link_method='oauth'`
  - **KHÔNG update SĐT/Email/CCCD** ở GG (V1 — chấp nhận data 2 bên lệch)
- Màn confirmation:
  - Title: "Liên kết thành công 🎉"
  - Hiển thị: tên Scalef username
  - Nút "Bắt đầu" → redirect về trang trước flow linking
- Unlock Phase 2: user giờ có `scalef_user_id` → FE Creator middleware cho phép join campaign

**V1 risk:**
- KHÔNG check `scalef_user_id` đã thuộc GG user khác → có thể duplicate (race condition rare)
- Data SĐT/Email/CCCD 2 bên có thể lệch → ảnh hưởng tracking commission nếu Scalef dùng SĐT làm key
- **V2 sẽ fix:** add FR-004 (check 1-1) + FR-005-018 (matching + conflict + OTP + saga)

**Full spec gốc:** [Account Linking FR-011](prd-account-linking-2026-05-04.md#fr-011-liên-kết-thành-công)

### FR-T2-013: Pending state resumable

**Source:** Account Linking PRD — FR-013
**Priority:** Should Have → **Promoted to V1** (UX critical)

**V1 simplification:** Pending state V1 chỉ track 1 step (sau callback OAuth, trước commit), không cần phức tạp như PRD gốc.

**Key Acceptance Criteria:**
- Collection `pending_scalef_link` schema (V1):
  - `user_id`, `scalef_user_id`, `scalef_snapshot` (user_name, phone, email), `step` (`oauth_callback_received` / `committed`), `created_at`
  - TTL 15 phút
  - Unique index `user_id`
- Resume logic:
  - User refresh browser sau OAuth callback → fetch pending state → resume từ step
  - Hết 15' → delete pending, redirect Step 1 (Settings)
- Cleanup khi link thành công (FR-T2-011) → delete pending

**Full spec gốc:** [Account Linking FR-013](prd-account-linking-2026-05-04.md#fr-013-pending-link-state-resumable)

> **Note V1:** Pending state V1 đơn giản hơn V2 (chỉ 2 step). Khi V2 add matching/conflict/OTP → mở rộng `step` enum.

### FR-T3-011: BE Proxy Scalef APIs

**Source:** FE Creator PRD — FR-011
**Priority:** Must Have (technical foundation)

## V1 Known Limitations (sẽ fix ở V2)

| Limitation | Impact | V2 fix |
|------------|--------|--------|
| Không check CCCD matching | Risk linking sai người | FR-005 (matching) + FR-018 (normalization) |
| Không check 1-1 (1 GG ↔ 1 Scalef) | Cùng Scalef account có thể link nhiều GG user | FR-004 |
| Không conflict resolution | Data SĐT/Email 2 bên có thể lệch | FR-006/007 (compare + chọn) |
| Không OTP verify khi update | Không update được profile cross-system | FR-008 |
| Không bidirectional update | Profile 2 bên không sync | FR-010 (saga) |
| Không trang my-links | Creator không xem lại được lịch sử link đã tạo | FR-006 (Browse my links) |
| Không browse all campaigns | Creator chỉ vào affiliate qua Event | FR-007 (Browse all) |
| Không commission dashboard | Creator không thấy được hoa hồng | FR-009 (Commission) |
| Không analytics tracking | Không đo funnel activation | FR-012 (Tracking events) |
| Không audit log đầy đủ | Dispute khó truy vết | FR-014 (Audit log) |
| Không reverse linking | Publisher Scalef legacy chưa migrate được | FR-015 (Reverse) + FR-016 (Batch) |
