# Cập nhật thông số nội dung thủ công

- **Các trường hợp cần cập nhật thủ công**
    - Hệ thống cập nhật sai thông số ⇒ Nhập tay lại thông số
    - Cập nhật thông số cho Video Facebook (hệ thống chưa hỗ trợ auto cập nhật lượt view)
- **Các bước cập nhật thông số nội dung thủ công**
    - **Bước 1:** Vào trang chủ ***“Nội dung”***  ⇒ Nhấn button ***“Update thông số”***
